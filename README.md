# ongil-medicall — 부모님 곁의 복약 동반자

부모님의 복약을 알림 전화와 스마트 약 관리로 함께 챙겨드리는 서비스 랜딩 페이지입니다.

## 파일 구조

```
.
├── index.html   # 메인 HTML
├── style.css    # 디자인 시스템 기반 스타일
└── README.md
```

## 디자인 시스템

Anthropic Claude Design Language 토큰을 기반으로 제작되었습니다.

- **폰트**: Cormorant Garamond (디스플레이) + Noto Sans KR (본문)
- **Primary**: `#cc785c` (코랄)
- **Canvas**: `#faf9f5` (크림)
- **Dark Surface**: `#181715` (다크 네이비)

## GitHub Pages 배포

1. 이 레포를 GitHub에 push

3. 저장 후 `https://<username>.github.io/ongil-medicall/` 접속
